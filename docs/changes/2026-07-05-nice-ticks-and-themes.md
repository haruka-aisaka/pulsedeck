# グラフ目盛りのキリのいい値化とライト/ダークテーマ対応

## 背景

- グラフの罫線がレンジの 4 等分位置に引かれるため、目盛りが 4.2k / 3.2k のような半端な値になる
- ダークテーマ固定のため、明るい環境や好みに合わせられない

## 現状の挙動

- Y 軸: 値域を単純 4 等分した位置に罫線とラベル
- X 軸: 表示ウィンドウを 4 等分した任意時刻にラベル
- テーマ: ダーク固定

## 変更後の挙動

- **Y 軸**: 1 / 2 / 2.5 / 5 × 10ⁿ 系列のキリのいい刻みで罫線・ラベルを引く（例: 0-100% は 25/50/75、
  動的レンジのネットワークは 2.5/5/7.5 MB/s など）
- **X 軸**: 時刻の切りのいい境界（15 秒・2 分・30 分・6 時間など、ウィンドウ幅に応じて選択）に
  ラベルを配置。ローカルタイムゾーンに整列
- **テーマ**: ヘッダー右にトグルボタンを新設し、auto → light → dark の 3 モードを循環
  - auto は OS 設定 (prefers-color-scheme) に追従し、OS 側の切り替えにも即時反応
  - 選択は localStorage に保存しリロード後も維持
  - グラフ・ゲージ・ステータスバー色 (theme-color) もテーマに追従

## スコープ外

- テーマのカスタムカラー設定
- グラフ目盛り数の変更（約 4 本のまま）

## E2E テストチェックリスト

- ### 正常系
  - [ ] Y 軸ラベルがキリのいい値になる（%・°C・MB/s 各グラフ）
  - [ ] X 軸ラベルが切りのいい時刻になる（1min/10min/3h の各レンジ）
  - [ ] トグルで light / dark が切り替わり、リロード後も維持される
- ### 異常系
  - [ ] localStorage が使えない環境でもエラーにならず auto で動作する
- ### エッジケース
  - [ ] ライトテーマでグラフの罫線・ラベル・ゲージ内円が視認できる
  - [ ] auto モード中に OS のテーマを切り替えると即時反映される
